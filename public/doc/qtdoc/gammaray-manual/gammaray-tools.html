    Tools

    Overview

    GammaRay consists of a number of specialized tools to inspect various aspects of Qt. You can select these using the tool selector on the left of the GammaRay window.

    User Interface Inspection

    The following tools can be used for insepecting graphical user interface issues, such as layouting, rendering, or event handling.

    • Qt Quick 2 Inspector
    • Widget Inspector
    • Graphics Scene Inspector
    • Qt3D Inspector
    • Styles
    • Web Inspector

    Application Behavior Inspection

    The following tools can be used to obeserve application behavior at runtime.

    • State Machine Debugger
    • Messages
    • Signal Plotter
    • Timers
    • Wayland Compositors
    • Script Engine Debugger

    Model and Data Structure Inspection

    The following tools allow you to inspect and manipulate models and data structures of your application at runtime.

    • Object Browser
    • Actions
    • Models
    • Text Documents
    • Resources
    • Translations
    • Meta Object Browser
    • Meta Type Browser
    • QtIvi Inspector

    System Information

    The following tools allow you to inspect information of the Qt installation and the system it's deployed on.

    • Font Browser
    • Locales
    • Mime Types
    • Network
    • Text Codecs
    • Standard Paths
