    Class Info

    Overview

    The class info view is available for all QObject and QMetaObject instances and shows per-class information defined in the corresponding QMetaClassInfo instance.

    The class info view shows the following information:

    • The name of the class information.
    • The value of the class information.
    • The class or namespace in which the class information has been declared.
