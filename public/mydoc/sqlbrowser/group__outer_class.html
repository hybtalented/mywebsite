添加一个数据库连接

参数
[in]driver数据库驱动 类型 字符串 可以为以下值: QSQLITE sqlite数据库 QMYSQL mysql数据库 QODBC odbc数据库 QPSQL postgreSQL数据库
[in]dbName数据库名 类型 字符串 对于QSQLITE为文件路径，对于其他数据库为数据库的名称
host数据库主机 类型 字符串 主机ip，对于QSQLITE可以不填
[in]user用户名 类型 字符串
[in]passwd密码 类型 字符串
[in]port端口号 类型 字符串 默认值 -1
返回
数据库发生的错误
